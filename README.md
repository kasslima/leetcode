# LeetCode Solutions

This repository is a personal collection of my solutions to various [LeetCode](https://leetcode.com/) problems. 

## 📁 Structure

The repository is organized by problem difficulty. Inside each difficulty directory, you'll find folders named after the specific problem, containing the solution files.

```
.
├── Easy/
│   ├── Longest-Common-Prefix/
│   ├── Palindrome Number/
│   └── Roman to Integer/
├── Medium/
│   └── Add Two Numbers/
└── Hard/
    └── ...
```

## 💻 Language

The primary programming language used for these solutions is **Go**.

## 🚀 Running the Solutions

To run a specific solution, navigate to its directory and use the `go run` command. For example:

```bash
cd "Medium/Add Two Numbers"
go run main.go
```
